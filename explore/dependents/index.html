Setup Guide

Upload your data

Upload a GitHub Action artifact .zip or a standalone dependency graph .json file.

Include graph data

The dashboard looks for dependency_graph.json, then falls back to the first non-SPDX .json file in the archive.

Use nodes and edges

Include top-level nodes and edges arrays with repository metadata inside each node's data object.

Add SPDX snippets

Add data.snippetPath for each node when a downloadable package SPDX snippet is available.
