Project Dependent Visualizations

Dependency Graph Visualizer

Explore software supply chains, discover dependents, and generate SPDX manifests.

Explore Dependents

View downstream consumers in a sortable data table or visualize them in an interactive network graph.

SPDX Generation

Click any row or node to extract valid SPDX 2.3 snippets securely pinned to specific commit SHAs.

Rich Metadata

Automatically aggregates repository stars, commit history, contributor counts, and academic citations.

How to Use the Dashboard

  • Upload Data: Use the control panel to upload the .zip artifact generated by the GitHub Action.
  • List View (Default): Sort consumers by stars, contributors, or activity. Click any row to view its SPDX manifest.
  • Graph View: Toggle to the Graph display to visually map deep, multi-level dependency chains.
  • Prune & Filter: Use the Depth slider to filter out indirect dependencies and focus on direct (L1) consumers.
